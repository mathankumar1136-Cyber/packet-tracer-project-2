# Enterprise Network Design (Cisco Packet Tracer)

## 📌 Project Overview
This project demonstrates a scalable Enterprise Network design using Cisco Packet Tracer. The network is designed to provide secure, efficient, and organized communication between multiple departments in an organization.

## 🏗️ Network Design Features
- VLAN segmentation for department-wise traffic separation
- Inter-VLAN Routing for communication between VLANs
- DHCP for automatic IP address assignment
- Dynamic Routing (OSPF/EIGRP if used) for efficient routing
- Secure remote access using SSH

## 🔧 Technologies Used
- Cisco Packet Tracer
- VLAN Configuration
- Router-on-a-Stick / Inter-VLAN Routing
- DHCP Server Configuration
- OSPF / EIGRP (if implemented)
- SSH Security Setup

## 🎯 Objectives
- To design a real-world enterprise network structure
- To improve network security using segmentation (VLANs)
- To enable efficient routing between networks
- To automate IP configuration using DHCP

## 🧠 Learning Outcomes
- Practical understanding of enterprise-level network design
- Hands-on experience in routing and switching
- Improved knowledge of VLANs, DHCP, and routing protocols
- Network troubleshooting and configuration skills
